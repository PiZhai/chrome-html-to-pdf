HTML转PDF测试页面

这是一个用于测试HTML转PDF功能的演示页面。该页面包含了各种常见的HTML元素，用于验证转换效果。

项目功能说明

本项目通过WebSocket连接Chrome浏览器的开发工具协议(Chrome DevTools Protocol)，实现了将HTML页面转换为PDF文件的功能。主要流程包括：

  1. 查找和启动Chrome浏览器
  2. 建立WebSocket连接
  3. 启用Page域
  4. 导航到本地HTML文件
  5. 等待页面加载完成
  6. 生成PDF文件
